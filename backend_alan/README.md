# FRIDAY (Project ALAN) - Backend

This is the backend agent for FRIDAY, operating on the Project ALAN architecture.
It uses:
- **LiveKit Agents**: For real-time voice and video.
- **Gemini 2.0 Flash Exp**: For low-latency speech-to-speech.
- **Python 3.11**: As the runtime environment.

## Deployment
This Space is configured with a Docker SDK. It runs `agent.py` on startup.
